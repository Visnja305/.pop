Brand Application Rules

Rules governing how the visual system is applied to digital layouts to guarantee consistency.

Visual Consistency

A brand system is defined by its boundaries. By adhering strictly to these layout parameters, Pop AI projects maintain an elite, cohesive presence across all platforms and collateral.

Color Allocation Rule

  • Maintain strict Dark Mode at all times. Canvas must use Obsidian Black (#050808).
  • Utilize secondary dark container shapes (#111413) to divide screen layouts.
  • Apply the primary Mint Glow (#0CC481) selectively for highlighting key features, active button actions, or gradients. Never exceed 10% accent presence.

Typography & Text Architecture

  • Display headings exclusively in Outfit with bold weights and tight letter spacing.
  • Format buttons, nav links, and metadata tags in uppercase with 0.08em wide tracking.
  • Set paragraphs in Inter using the Warm Cream text color at 1.6 line height.

Visual Mockups & Image Rules

  • Use only clean UI mockups, terminal frames with code snippets, and thin geometric charts.
  • Strictly avoid generic AI stock images (like blue network lines, brains, or humanoid robots).
